📌 三个步骤完成训练

① 准备好 data/my_romance_data.jsonl 训练数据
② 打开终端，输入一行命令开始训练
③ 等待 15-30 分钟，查看拟合报告

第 1 步：认识你的文件

这个项目里有两个重要的东西：

📄 训练数据文件
位置：data/my_romance_data.jsonl

里面装了 20 条恋爱对话的"标准答案"。每条包含玩家说的话和小棠应该怎么回复。
类比：就像给小棠买了一本《恋爱对话练习册》，里面每道题都有参考答案。

🖥️ 训练脚本
位置：scripts/alignment_tuner.py

这个脚本会"教"小棠学习恋爱对话。
类比：就像一位老师，拿着练习册一道道教小棠怎么回答。

第 2 步：打开终端

1
找到终端程序
Mac 电脑：按键盘上的 Cmd + 空格，输入"终端"，回车
Windows 电脑：按 Win 键，输入 cmd，回车
2
进入项目文件夹
在终端里输入（把 /你的项目路径 换成实际位置）： 
cd /你的项目路径/heartscape-engine
💡 如果不确定路径，可以把文件夹直接拖进终端窗口
3
确认数据文件存在
在终端里输入： 
ls data/my_romance_data.jsonl
如果显示文件路径，说明数据找得到。如果显示 "No such file"，检查文件是否在正确位置。

第 3 步：开始训练！

在终端里输入这一行命令，然后按回车：

python scripts/alignment_tuner.py --data data/my_romance_data.jsonl --interactive
🔍 这条命令在说什么？
--data = 用哪个练习册（数据文件）
--interactive = 训练完后可以和小棠对话测试

你会看到屏幕上出现这些内容：

屏幕上显示什么什么意思

⚠️ 注意：训练需要 15-30 分钟，期间电脑风扇可能会转得快一些，这是正常的。
不要关掉终端窗口，让它跑完。

第 4 步：看懂成绩单

训练结束后会显示一份"拟合报告"，长这样：

指标训练前训练后变化
语义相似度0.3500.780↑ +0.430
Token重叠度0.1200.340↑ +0.220
训练 Loss—0.5123↓ 下降 58%
怎么看懂这些数字？
  • 语义相似度：0 是完全不像，1 是一模一样。训练后从 0.35 涨到 0.78，说明小棠学会了一大半。
  • Token重叠度：回答和标准答案用了多少相同的词。涨了就是进步。
  • Loss：错误率。越小越好，下降 58% 说明学得很好。
✨ 什么样的结果是好的？
语义相似度 > 0.70 = 优秀（小棠学得很像）
语义相似度 0.50-0.70 = 良好（还需要再练练）
语义相似度 < 0.50 = 可以增加数据量再训练一次

🔧 高级：调整训练参数

如果觉得训练效果不够好，可以调整这些参数。在命令后面加上就行：

参数默认值调大效果调小效果
--r32学得更细，但更慢学得更快，但可能不够
--lr5e-5学得快，但可能"记错"学得慢，但更稳
--epochs5反复多练几遍少练几遍

例如，想让小棠学得更"用功"：

python scripts/alignment_tuner.py --data data/my_romance_data.jsonl --r 32 --lr 5e-5 --epochs 5 --interactive

❓ 常见问题

Q：训练到一半电脑卡住了怎么办？
A：不是卡住了，是在计算。耐心等，不要关窗口。如果超过 1 小时还没反应，可能是显存不够，尝试加 --epochs 3 减少训练量。
Q：提示 "No module named transformers"？
A：还没安装依赖。在终端输入 pip install -e ".[dev,train]" 然后再试。
Q：训练完的模型在哪里？
A：在 lora_alignment/ 文件夹里。里面有 adapter_model.safetensors 文件，这就是训练好的"小棠恋爱脑"。
Q：我可以自己写训练数据吗？
A：当然！打开 data/my_romance_data.jsonl，照葫芦画瓢添加新的对话。每一行是一个 JSON 对象，格式是：
{"text": "<|im_start|>user\n你说的...<|im_end|>\n<|im_start|>assistant\n小棠回...<|im_end|>"}
Q：能用网页操作吗？
A：可以！在终端输入 python scripts/web_ui.py，然后用浏览器打开 http://localhost:8765，就能在网页上点点点完成训练。
